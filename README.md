# 🏠 House Price Prediction 
A Machine Learning-based web application that predicts house prices based on user-provided property details. This project uses Python, Flask, and Machine Learning algorithms to estimate the market value of a house. 

## 📌 Features 
- Predict house prices instantly.
- User-friendly web interface.
- Machine Learning-based prediction model.
- Real-time input and output.
- Responsive design using HTML and CSS.

## 📊 How It Works
- User enters house details.
- Input data is processed by the machine learning model.
- The model predicts the estimated house price.
- Predicted price is displayed on the webpage.
